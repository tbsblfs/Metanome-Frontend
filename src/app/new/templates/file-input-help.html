File Input Help


If you want to add new files to the available files, add your files to the folder "/WEB-INF/classes/inputData/". CSV and TSV files are accepted by Metanome. The files need to be UTF-8 or ASCII encoded.


Additional settings:
Do not escape any characters, Metanome is taking care of that. For example, if you want a backslash as escape character, just write a single "\\". For tab separator type in "\\t". For the null string you can either type in "\\0" or just leave the field empty.


Make sure, that you set the correct settings, so that your file input can be read. If a line could not be read properly, null is returned. Internally, au.com.bytecode.opencsv.CSVReader is used to read the input file.
